# 주간보고 (5/18 ~ 5/24)

**프로젝트**: 사내 데이터분석 플랫폼 구축

## 한 줄 요약
설계부터 실제 작동하는 초기 버전까지 1주 만에 완료했습니다.

## 이번 주 한 일

**1. 무엇을 만들지 확정 (월요일)**
- 사용자가 플랫폼에서 할 수 있는 일 40가지 정리
- 만들어야 할 기능 7덩어리로 나눠 작업 계획 수립

**2. 실제 작동하는 시스템 구축 (월~금)**
- 분석가, 관리자, 감사담당자, 일반조회자 등 역할별 화면 4종 제작
- 데이터 조회·분석·기록 등 핵심 기능 모두 구현

**3. 직접 사용해보며 3차례 점검 통과 (금)**
- 데이터 조회 시 주민번호·이름·전화번호 자동 가림 처리 확인
- 분석가가 데이터 불러와 결과 보는 과정 자동 테스트로 검증
- **"질문을 한국어로 입력하면 AI가 분석 코드를 만들어 분석 화면에 자동 입력"** 핵심 기능이 10초 안에 완료되는 것 확인

## 의미

- 단순 시연용이 아니라 **실제 데이터와 실제 AI로 작동**하는 것을 확인했습니다.
- 누가 무엇을 조회했는지 모든 기록이 남아 보안·감사 기준을 충족합니다.
- 회사가 원했던 "AI가 도와주는 데이터 분석" 기능이 실제로 동작합니다.

## 다음 주 계획

- 사내 계정으로 로그인하는 기능 연결
- 많은 사람이 동시에 써도 괜찮은지 부하 점검
- 외부망이 차단된 환경에서도 작동하는지 확인
