## ⚠️ 모호점 1: MVP 범위와 기능 충돌

### 상황
- **Q27**: 1~2개월 내 MVP 출시 희망
- **Q3**: RBAC 권한 분리 필요 (관리자/분석가/뷰어)
- **Q29**: Security Baseline 강제 적용
- **Q25**: 사용자 활동 전수 감사 로깅 필요
- **Q26**: PII 자동 마스킹 필요
- **Q28 우선순위**: 1(DB+SQL), 2(파일분석), 3(차트), 6(노트북) — **인증/권한(5), 보고서(4), 공유(7), 스케줄링(8) 제외**

### 문제점
1~2개월 일정에는 Q28 우선순위 4개(DB+SQL, 파일, 차트, 노트북)만 들어가는 게 현실적입니다. 그런데 **인증/권한이 빠지면 RBAC, 보안 기준, 감사 로깅, PII 마스킹이 작동할 토대가 없습니다**.

### Clarification Question 1
MVP(1~2개월) 범위를 어떻게 잡으시겠습니까?

A) **최소 보안 포함 MVP**: 1(DB+SQL), 2(파일), 3(차트), 6(노트북) + **최소 인증(로그인) + 감사 로그 기본 + PII 마스킹 기본** (보고서/공유/스케줄링은 Phase 2)
B) **PoC 우선 MVP**: 1, 2, 3, 6만 (인증/감사/마스킹 없이) — 사내 폐쇄 PoC 후 Phase 2에서 보안 기능 통합
C) **MVP 일정 연장**: 1~2개월은 무리, 3~6개월로 늘리고 5(인증/권한) + 4(보고서) 포함
D) Other (please describe after [Answer]: tag below)

[Answer]: A

## ⚠️ 모호점 4: 외부 LLM 모델 아키텍처

### 상황
- **Q31**: "fabi.ai가 온프레미스로 가는 느낌이면 좋겠음. 내부망은 API에서, 모델은 외부에 띄우는 방식으로 하면 보안과 속도를 다 잡을 수 있지 않을까."
- **Q18**: 제한적 외부망 (특정 도메인 화이트리스트)
- **Q25**: 사용자 활동 전수 감사
- **Q26**: PII 자동 마스킹

### 문제점
"모델을 외부에 띄운다"는 의미와, 그 모델이 무엇을 위한 것인지 명확화가 필요합니다. 또한 **내부 데이터(스키마/쿼리/샘플)가 외부 LLM API로 전송되는 시점**의 보안 정책을 정해야 합니다.

### Clarification Question 4-1
외부 모델(LLM)을 사용하는 주요 기능은 무엇인가요? (여러 개 가능, 콤마 구분)

1) **자연어 → SQL 변환** (Text-to-SQL)
2) **차트 자동 추천** (데이터 보고 어떤 차트가 적합한지)
3) **데이터 요약 / 인사이트 추출** (자동 보고서 코멘트)
4) **코드 어시스턴트** (Python/R 코드 생성 도움)
5) **데이터 품질 검토** (이상치, 결측치 자동 탐지 설명)
6) 아직 모르겠음 — 일단 1번부터 시작

[Answer]:1,3,4

### Clarification Question 4-2
외부 LLM API에 어떤 정보까지 전송 가능한가요?

A) **메타데이터만**: 스키마(테이블/컬럼 이름, 타입), 샘플 없음 — 가장 안전
B) **메타데이터 + 익명화 샘플**: 마스킹된 샘플 행 몇 개 포함
C) **실데이터 일부**: 비PII 컬럼은 원본, PII는 마스킹 후 전송
D) **아직 미정 — 보안팀과 협의 필요**, 일단 A로 시작
E) Other (please describe after [Answer]: tag below)

[Answer]: D

### Clarification Question 4-3
외부 모델 호스팅 방식은?

A) **상용 API** (OpenAI/Anthropic 등) — 사내 프록시 거쳐서 화이트리스트로 호출
B) **사내 GPU 서버에 오픈소스 모델** (Llama 등 셀프 호스팅) — 외부 송신 없음, 다만 "외부"가 아니라 별도 사내 영역
C) **클라우드 자체 호스팅** (사내 클라우드 계정에 격리된 VPC) — Q31의 "외부" 의미에 부합
D) **아직 미정 — 추천 부탁**
E) Other (please describe after [Answer]: tag below)
